Course 1: ASP.NET Core Essentials

Master key ASP.NET Core features and patterns

Dependency injection, DTOs, minimal APIs, the configuration system, and more.

Learn not just how, but when and why to use them.

Get solid REST API development foundations

Master the key principles and conventions behind RESTful API development so you can build APIs that are scalable, flexible, and easy to use from any client.

Use Vertical Slice Architecture

Leave behind the traditional layered architecture and embrace a feature-based approach to build maintainable and scalable applications.

Access your database via Entity Framework Core

Use the popular Entity Framework Core O/RM framework to interact with your relational database without writing a single line of SQL. It's all C#.

Test it with an end-to-end user experience

Make your backend API shine by integrating it with the included Blazor frontend, so you can see how your API works in a real-world scenario.
